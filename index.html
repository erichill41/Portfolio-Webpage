About Me

I started my career in sales after earning a Bachelor's Degree in Marketing from the University of Central Oklahoma. With a background in business and technology sales, I aim to bridge the gap of disconnect between technical teams and the business management side of things in small-medium businesses. Portfolio

Local Library Project

The local library project works to create a library for a local neighborhood using JavaScript that gives users information about available books to checkout and most popular authors/books/genres. I built this to continue my practice of using higher order functions and supply neighborhoods with a way to track books.

  • Skills Used: JavaScript, higher order functions.

  • Live Page

Decoder Ring Project

The Decoder Ring Project works to encode and decode messages using JavaScript functions. It's useful for anyone wanting to send their friend an encrypted message, or to decode a message that was received. I built this to further solidify skills in JavaScript and working with complex algorithms, as well as giving users a fun way to encrypt messages.

  • Skills Used: JavaScript, helper and higher order functions.
